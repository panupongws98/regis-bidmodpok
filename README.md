# Drag Bike Registration Desk

เว็บแอพแบบ local สำหรับบันทึก จัดเก็บ ค้นหา แก้ไข ลบ และพิมพ์ข้อมูลผู้สมัครแข่งขันแด็กไบค์ โดยข้อมูลจะถูกเก็บไว้ในไฟล์ JSON ภายในเครื่อง

## วิธีใช้งาน

1. เปิด PowerShell ที่โฟลเดอร์โปรเจกต์นี้
2. รันคำสั่ง

```powershell
node server.js
```

3. เปิดเบราว์เซอร์ไปที่ [http://127.0.0.1:3000](http://127.0.0.1:3000)

## ความสามารถหลัก

- บันทึกข้อมูลชื่อผู้สมัคร ที่อยู่ รุ่นแข่งขัน จำนวนรถ และหมายเลขรถแต่ละคัน
- ตรวจสอบหมายเลขรถซ้ำกับข้อมูลเดิมอัตโนมัติ
- ค้นหา แก้ไข ลบ และคัดลอกหมายเลขรถได้
- จัดการรายการรุ่นแข่งขันผ่านหน้าเว็บ
- พิมพ์ใบสมัครรายรายการ หรือพิมพ์รายการรวมทั้งหมดได้
- ตั้งค่าได้ว่าตารางพิมพ์รายการรวมจะแสดงคอลัมน์ใดบ้าง และเรียงลำดับคอลัมน์อย่างไร

## ตำแหน่งไฟล์ข้อมูล

- ข้อมูลผู้สมัคร: `data/registrations.json`
- รายการรุ่นแข่งขัน: `data/settings.json`
